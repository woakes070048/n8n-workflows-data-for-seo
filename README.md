## ‚öôÔ∏è How to Use

Each workflow folder in this repository contains its own `.json` file and a specific `README.md` with detailed setup instructions. The general process is simple:

1.  **Download the Workflow:** Navigate to the folder for the workflow you want to use and download the `.json` file.
2.  **Import into n8n:** In your n8n instance, choose "Import from File" and select the `.json` file to create the new workflow on your canvas.
3.  **Configure Credentials & Settings:** Follow the instructions in that workflow's specific `README.md` to connect your own API credentials (e.g., for Google, DataForSEO, etc.) and configure any required settings.
4.  **Activate & Run:** Activate the workflow and start automating!
